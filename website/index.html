WHY AIVECTORMEMORY

别人做记忆，我们做工作流

其他记忆工具只管存取。AIVectorMemory 让 AI 自动管理完整开发流程。

独有

问题追踪 track

发现 bug → 排查根因 → 修复 → 测试 → 归档。完整生命周期管理，AI 自动记录每一步进展，归档后自动提取经验。

$ track create "登录页500错误"
✓ 已创建问题 #42
$ track update --root-cause "..."
✓ 已更新根因分析
独有

任务管理 task

需求 → 设计 → 拆分子任务 → 逐个执行。多步骤新功能自动拆分，按顺序执行，进度实时同步。

T1 设计数据库 schema
T2 实现 API 端点
T3 前端页面开发进行中
T4 集成测试
独有

跨会话状态 status

阻塞状态、当前任务、待办事项、最近变更——切换会话永不丢失。新会话自动恢复上下文。

当前任务#300 官网重设计
阻塞状态等待确认
活跃问题3 个
待办任务12 / 15 完成
独有

Hooks 规则执行 hooks

bash_guard 拦截危险命令、stop_guard 阻断违规回复、check_track 强制问题记录。硬阻断，不靠 AI 自觉。

$ rm -rf /important-data
✗ BLOCKED by bash_guard
危险命令已拦截，需要用户手动执行

CAPABILITIES

核心能力

基础记忆能力同样领先，每一项都经过精心打磨。

语义搜索

向量相似度匹配，搜"数据库超时"也能找到"MySQL 连接池踩坑"。

省 50%+ Token

按需检索只加载相关记忆，不再批量注入上下文。省钱又精准。

GET STARTED

30 秒开始使用

两行命令，为你的所有 IDE 装上记忆。

Terminal
$pip install aivectormemory
$avmrun install
✓ 已自动为 Cursor、Claude Code、VSCode 等 10 款 IDE 配置 MCP
$avmrun web# 启动 Web 数据看板
→ http://localhost:9080

或使用 uvx（零安装试用）

$ uvx aivectormemory install
